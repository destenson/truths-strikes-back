# truths-strikes-back
Trump's Truths in the style of Star Wars

## Run locally

Open `index.html` in a browser.

The page tries to load the latest posts from `@realDonaldTrump` on Truth Social via RSS feeds (including JSON feed adapters for browser compatibility), and presents them in a Star Wars-style crawl over a moving starfield. It uses third-party adapter/proxy services (`api.rss2json.com` and `api.allorigins.win`), which receive the RSS URL and your IP address, and may be subject to their rate limits or availability.
